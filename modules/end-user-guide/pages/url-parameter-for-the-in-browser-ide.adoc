= URL parameter for the in-browser IDE

If the URL for starting a new workspace doesn't contain a URL parameter specifying the integrated development environment (IDE), the workspace loads with the default in-browser IDE, which is Microsoft Visual Studio Code - Open Source.

The URL parameter for specifying another supported in-browser IDE is `che-editor=__<editor_key>__`:

[source,subs="+quotes,+attributes,+macros"]
----
pass:c,a,q[{prod-url}]#__<git_repository_url>__?che-editor=__<editor_key>__
----

This URL parameter enables a user to start a workspace with a clone of the same project repository and the personal choice of the in-browser IDE.

NOTE: The in-browser IDE might be already set for a remote Git repository in the xref:defining-a-common-ide.adoc[`che-editor.yaml` file of the repository].
